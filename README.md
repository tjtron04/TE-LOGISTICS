# TE Logistics - Official Website

**Fast & Reliable Air Freight Logistics Solutions**

## About

TE Logistics is a professional logistics company based in Bangalore, India, specializing in air freight and cargo transportation. This is the official website showcasing our services and expertise.

## Features

- ✈️ **International Air Freight Services**
- 📦 **Domestic Air Cargo Solutions**
- 🚚 **Door Pickup Service**
- 🏢 **Airport Cargo Handling**
- 📋 **Export and Import Logistics Support**
- ⚡ **Express Cargo Shipping**

## Industries Served

- Electronics
- Pharmaceutical Products
- Automotive Spare Parts
- Garments and Textiles
- E-commerce Shipments
- Industrial Equipment

## Website Structure

- `index.html` - Main website page with all sections
- `styles.css` - Professional styling with red and black color scheme
- `script.js` - Interactive features (smooth scrolling, form handling)
- `logo.png` - TE Logistics logo

## Getting Started

1. Clone the repository
2. Add the logo.png file to the root directory
3. Open index.html in your browser to view the website
4. Customize content as needed

## Deployment

To deploy to GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages section
3. Select main branch as source
4. Save and your site will be live
